# Создание сети с Выделенным сервером

Dedicated Server -> dedicated network -> dedicated subnet

## Входные параметры

* project_id - уникальный идентификатор связанного проекта.
* price_plan_name - название тарифного плана. 
* region_name - имя пула в котором будет создан выделенный сервер.
* subnet_cidr1 - подсеть в локальной сети выделенного сервера
