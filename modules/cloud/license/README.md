# License

Terraform module for creating a single Selectel Cloud License.

## Variables

  * `license_project_id` - Project of the Selectel Cloud License.

  * `license_region` - Region of the Selectel Cloud License.

  * `license_type` - Type of the Selectel Cloud License.

## Outputs

  * `license_id` - ID of the created Selectel Cloud License.
