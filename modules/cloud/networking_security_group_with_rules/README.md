# Модуль создания Openstack группы безопаности и правил

Модуль создания Openstack группы безопсности и правил для послежующей ассоциации с портом:
- группа безопасности
- десять примеров распространенных правил

## Принимаемые переменные

  * `name` - Имя создаваемой группы безопасности.


## Выходные переменные

  * `networking_secgroup_id` - UUID cозданной группы безопасности
