# VRRP Subnet

Terraform module for creating a single Selectel Cloud VRRP Subnet.

## Variables

  * `vrrp_subnet_project_id` - Project of the Selectel Cloud VRRP Subnet.

  * `vrrp_subnet_region` - Region of the Selectel Cloud VRRP Subnet.

  * `vrrp_subnet_prefix_length` - Prefix length of the Selectel Cloud VRRP Subnet.

  * `vrrp_subnet_ip_version` - IP version of the Selectel Cloud VRRP Subnet.

## Outputs

  * `vrrp_subnet_id` - ID of the created Selectel Cloud VRRP Subnet.
